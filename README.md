# 学生管理系统（SSM 版）
基于 SSM（Spring+SpringMVC+MyBatis） 框架开发的 Web 端学生管理系统，实现学生信息的增删改查、分页查询、条件筛选等核心功能，新增性别精准查询、年龄区间筛选 两大特色功能，完善登录校验、权限控制、异常处理等基础能力，适配课程管理、公告展示等拓展功能，是入门 SSM 框架的实战项目。
## 一、项目简介
本项目是面向高校 / 培训机构的轻量级学生管理系统，基于 Java Web 技术栈和 SSM 分层架构开发，遵循MVC 设计模式和分层开发思想（Controller/Service/DAO），实现学生信息的全生命周期管理。在基础的学生增删改查、分页查询、姓名模糊查询功能上，新增 2 个核心特色功能：
性别精准查询：支持「男 / 女 / 全部」性别筛选，带登录校验，筛选结果实时回显；
年龄区间筛选：自定义「青年（10-30 岁）/ 中年（31-60 岁）/ 老年（60 岁以上）」年龄段，基于 MyBatis SQL 实现数据库层精准筛选，避免内存层数据过滤，提升查询效率。
项目完善了异常处理（如参数格式错误、学号不存在、未登录访问拦截）、结果回显、页面跳转等细节，贴合实际开发中的业务场景。

# 三、核心功能列表
### 基础核心功能（原有）
1. 学生信息管理：注册、登录 / 登出、密码修改、学生信息 增删改查；
2. 学生查询：按学号单查、姓名模糊查询、所有学生分页查询（每页 5 条）；
3. 课程管理：课程新增、删除、按编号查询、分页查询；
4. 选课管理：学生个人对课程的选课、退课，总览个人选择了几门课程，以及与当期课表的总课程对比；
5. 基础能力：公告展示、登录校验（未登录拦截）、异常处理（参数错误 / 数据不存在提示）、结果回显。
### 新增特色功能（重点实现）
| 新增功能 | 功能详情 |
|:--------|:---------|
| 性别精准查询 | 1. 支持「男 / 女 / 全部」三个选项筛选；<br>2. 带登录校验，未登录跳转至登录页；<br>3. 筛选结果以表格形式回显，无数据时提示；<br>4. 适配异常参数拦截，非指定选项给出错误提示。 |
| 年龄区间筛选 | 1. 自定义年龄段：青年（10-30 岁）/ 中年（31-60 岁）/ 老年（60 岁以上）；<br>2. 基于 MyBatis SQL 在数据库层实现区间筛选，提升效率；<br>3. 下拉框选择后提交，结果实时回显；<br>4. 保留选中状态，查询后下拉框仍显示当前选择的年龄段。 |

## 六、注意事项
1. 版本适配：项目基于 Tomcat7.0.96 开发，高版本 Tomcat 可能存在配置兼容问题，建议使用指定版本；
2. 路径大小写：SpringMVC 请求映射、JSP 文件路径严格区分大小写（如/toAge不可写为/toage，age.jsp不可含空格）；
3. 登录校验：所有功能页面均带登录校验，未登录访问会自动跳转到登录页，需先登录再操作；
4. 异常处理：所有用户输入（如学号、年龄）均做格式校验，非数字、空值、非法参数均会给出明确的页面提示；
5. 数据层筛选：年龄 / 性别筛选均在 DAO 层通过 SQL 实现，而非内存层过滤，提升大数据量下的查询效率；
6. 依赖问题：若启动时出现依赖缺失，需确认 Maven 依赖（或手动导入 jar 包）包含 SSM、MySQL 驱动、JSP 相关包。

```plain
